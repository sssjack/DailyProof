# DailyProof 部署文档

## 服务器

- IP：`8.141.121.133`
- 域名：`http://www.clockwise.asia/`
- 访问路径：`http://www.clockwise.asia/DailyProof`

## 部署步骤

1. 上传项目到 `/opt/dailyproof`。
2. 创建 `.env`，配置 PostgreSQL、JWT、DeepSeek。
3. 执行 `docker compose up -d --build`。
4. 在 Nginx 中增加 `/DailyProof` 反代到 `127.0.0.1:8091`。
5. 执行 `nginx -t && systemctl reload nginx`。
6. 请求 `/DailyProof/api/health` 和 `/DailyProof` 验证。

## 默认账号

- 管理员：`admin@dailyproof.cn` / `DailyProof@2026`
- 体验用户：`demo@dailyproof.cn` / `Demo@2026`

## 运维命令

```bash
cd /opt/dailyproof
docker compose ps
docker compose logs -f app
```
